**Steps:**
1. Run Engleג€“Granger `coint()` on all ${9 \choose 2} = 36$ pair combinations
2. Keep pairs with p-value < 0.05
3. Select best pair (lowest p-value)
4. Estimate hedge ratio $\hat{\beta}$ via OLS
5. Compute spread and 30-day rolling z-score

## Z-Score Signal Design

$$
z_t = \frac{S_t - \bar{S}_{30}}{\sigma_{S,30}}
$$

| Level | Signal |
|-------|--------|
| $z_t > +2$ | Spread is rich ג€” consider **short spread** |
| $z_t < -2$ | Spread is cheap ג€” consider **long spread** |
| $z_t \to 0$ | Mean-reversion target ג€” **exit** |
